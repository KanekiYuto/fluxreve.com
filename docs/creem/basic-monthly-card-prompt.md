# Basic Plan 营销卡片 - AI 生成提示词

## AI 图像生成提示词

```
Create a modern, professional marketing card design for a "Basic Plan - Monthly Subscription" pricing tier. The card should feature:

Visual Style:
- Clean, minimalist design with a gradient background (soft purple to blue)
- Modern tech/SaaS aesthetic
- Professional but approachable feel
- High contrast for readability

Layout Elements:
- "BASIC PLAN" badge/label at the top
- Large, bold "$15/month" pricing display in the center
- "1,500 Credits/Month" highlighted prominently
- Icon-based feature list showing:
  * AI models icon
  * High resolution (1024x1024) icon
  * Priority queue icon
  * Commercial use checkmark icon
  * Support icon

Design Details:
- Rounded corners with subtle shadows for depth
- Gradient border with a slight glow effect
- Icons should be simple, line-based, and consistent
- Typography: Sans-serif, modern font (similar to Inter or Poppins)
- Color scheme: Primary purple (#6633FF), secondary blue (#4A90E2), white text

Bottom Section:
- "Get Started" call-to-action button with hover effect
- Small text: "Perfect for individuals & small projects"

Overall mood: Professional, trustworthy, modern, and inviting
Aspect ratio: 3:4 (portrait card format)
Style: Flat design with subtle 3D elements
```

## 简化版提示词

```
Modern SaaS pricing card for "Basic Plan - $15/month". Clean gradient background (purple to blue), bold pricing display, icon-based feature list showing 1,500 credits, AI models, high resolution, priority queue. Professional minimalist design with rounded corners, soft shadows, and "Get Started" CTA button. Style: flat design, tech/startup aesthetic.
```

## 推荐使用的 AI 模型

- **DALL-E 3** 或 **MidJourney** - 适合高质量结果
- **Stable Diffusion XL** - 更多控制选项
- **Ideogram** - 适合文字较多的设计

## 获得最佳效果的技巧

1. 使用 HEX 色码指定精确颜色,保持品牌一致性
2. 如果想要纯设计卡片,添加 "no people" (无人物)
3. 添加 "professional product photography lighting" (专业产品摄影灯光) 增加深度
4. 使用负面提示词避免: "blurry, low quality, distorted text, cluttered" (模糊、低质量、扭曲文字、杂乱)
